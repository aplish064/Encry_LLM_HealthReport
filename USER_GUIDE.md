# 10模态健康监测系统使用指南

## 🎯 系统功能

这是一个隐私保护的多模态健康监测系统，支持10种数据模态的同态加密推理。

### 支持的模态

| 模态 | 类型 | 功能描述 |
|------|------|----------|
| 🛏️ **Depth Camera** | 图像 | 睡眠姿态检测 |
| 📡 **UWB Radar** | 时序 | 心率、血压监测 |
| 🏃 **IMU Sensor** | 时序 | 步态分析、代谢评估 |
| 📶 **WiFi CSI** | 时序 | 心率、呼吸监测 |
| 📷 **RGB Camera** | 图像 | 风险评分、跌倒检测 |
| 🦴 **NTU RGB+D** | 骨骼 | 动作识别、行为分析 |
| 👁️ **RetinaMNIST** | 医学图像 | 心血管疾病早期预警 |
| 🫁 **ChestMNIST** | 医学图像 | 肺部疾病筛查 |
| 🔬 **PathMNIST** | 医学图像 | 癌症筛查 |
| 🩸 **BloodMNIST** | 医学图像 | 血液疾病诊断 |

## 🚀 快速开始

### 1. 启动后端服务器

```bash
# 激活虚拟环境
source venv/bin/activate

# 启动后端（端口8082）
cd backend
uvicorn simple_app:app --host 127.0.0.1 --port 8082
```

### 2. 启动前端服务器

```bash
# 在新终端窗口中
cd frontend
python -m http.server 8001
```

### 3. 访问系统

打开浏览器访问: `http://127.0.0.1:8001`

## 📖 使用步骤

### Step 1: 选择数据模态

1. **浏览可用模态**: 每个卡片显示模态的数据可视化预览
2. **选择模态**: 点击卡片选择1-10种任意模态
3. **查看工具**: Step 2会自动更新显示对应的同态加密工具

### Step 2: 同态加密推理

- **DeepSeek智能调度**: LLM根据模态类型自动分配工具
- **并行加密推理**: 多个模态同时处理，保护数据隐私
- **实时进度显示**: 加密和处理过程可视化

### Step 3: 临床报告生成

- **健康指标**: 各模态分析结果
- **风险评估**: 综合健康评分
- **建议措施**: 个性化健康建议

## 🔒 隐私保护特性

- ✅ **本地加密**: 所有数据在本地加密后传输
- ✅ **密文计算**: 服务器只处理加密数据，无法查看原始内容
- ✅ **同态推理**: 直接在密文上进行机器学习推理
- ✅ **结果解密**: 推理结果返回本地后解密显示

## 🛠️ 故障排除

### 问题1: 模态卡片显示图标而不是缩略图

**原因**: 后端服务器未启动或数据文件缺失

**解决**:
```bash
# 检查服务器是否运行
curl http://127.0.0.1:8082/api/health

# 检查数据文件是否存在
ls -la test_data/
```

### 问题2: 点击"开始分析"后无响应

**原因**: 前端无法连接到后端API

**解决**:
1. 检查后端服务器是否运行在端口8082
2. 刷新浏览器页面 (Ctrl+F5)
3. 检查浏览器控制台是否有错误

### 问题3: Step 2工具集群没有显示

**原因**: 浏览器扩展干扰或JavaScript加载问题

**解决**:
1. 禁用浏览器扩展
2. 硬刷新页面 (Ctrl+Shift+R)
3. 清除浏览器缓存
